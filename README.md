## 🚀 Featured Projects

### 🗣️ turtlebot_llm_control — LLM-Powered Voice Interface for TurtleBot3
> A full ROS2 package that adds a **natural-language voice interface and LLM behaviour engine** to a TurtleBot3. Bridges offline speech recognition (faster-whisper), a locally-hosted LLM (Ollama/qwen2.5-coder), and Nav2 navigation into a single coherent pipeline.
>
> **Key features:**
> - 🎙️ Wake-word gated voice control ("Hey Pepper") with offline STT — no internet required at runtime
> - 🤖 LLM intent classification with rule-based fast path (eliminates ~500ms latency for safety-critical commands)
> - 🗺️ Tour-guide behaviour — records routes via teleop, replays with waypoint narration via TTS
> - 🧠 Behaviour tree orchestrator for navigation, recording, pause/resume, and localisation recovery
> - 😊 Emotion/gesture engine publishing to Pepper robot gesture controller in real-time
> - 🖥️ PyQt5 TSP-optimal waypoint GUI for tour planning
>
> **Stack:** `ROS2 Humble` · `Python 3.10` · `faster-whisper` · `Ollama` · `Nav2` · `PyQt5` · `Behaviour Trees`
>
> 📄 [Full Documentation](https://github.com/karanS08/turtlebot_LLM_control) · 🔗 [Repository](https://github.com/karanS08/turtlebot_LLM_control)
